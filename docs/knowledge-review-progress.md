# 지식 순차 점검 기록

트루핑키스 위키의 기존 지식을 `data/source/knowledge-order.json` 순서대로 점검한 기록입니다. 각 지식은 전체 JSON을 읽고 `유지 / 부분 수정 / 재구성` 중 하나로 판정한 뒤, 필요한 경우 본문·퀴즈·출처를 함께 확인합니다.

## 진행 기록

### 1. ai

- id: `ai`
- 파일: `data/source/knowledge/digital/ai.json`
- 판단: 부분 수정
- 본문 판단: 전체 범위는 적절하지만 `알아 두면 도움이 되는 이유`처럼 안내형 목차가 있고, AI 결과 확인 기준이 여러 곳에서 반복되었습니다.
- 퀴즈 판단: 핵심을 묻고 있으나 일부 오답이 “AI는 마음과 책임이 있다”처럼 바로 틀린 보기로 보여 선택지 품질을 더 높일 필요가 있었습니다.
- 출처 판단: 교육부 정책자료, 개인정보보호위원회, NIST 자료가 실제 확인 내용과 대응되어 유지했습니다.
- 수정 내용: 목차를 `AI의 뜻과 작동 방식`, `AI가 쓰이는 분야`, `AI 결과를 확인하는 기준`으로 재정리하고, 본문을 백과형 설명 흐름으로 다듬었습니다. 퀴즈는 작동 방식, 정보 확인, 개인정보 입력 기준을 묻도록 보기를 다시 구성했습니다.

### 2. data

- id: `data`
- 파일: `data/source/knowledge/digital/data.json`
- 판단: 부분 수정
- 본문 판단: 데이터의 뜻과 확인 기준은 적절했지만, `정보가 데이터가 되는 과정`처럼 목차가 추상적이고 같은 의미의 문장이 반복되었습니다. 데이터 역사 연표는 현재 출처 구성과 직접 대응이 약해 핵심 설명에서 제외하는 편이 낫다고 판단했습니다.
- 퀴즈 판단: 핵심을 묻고 있으나 질문 문장이 다소 넓어, 데이터의 뜻·좋은 데이터의 조건·숫자 데이터 확인 기준을 직접 묻도록 정리했습니다.
- 출처 판단: 통계청 KOSIS 통계놀이터와 개인정보보호위원회 자료는 데이터·통계·개인정보 설명의 사실 확인용 출처로 유지했습니다. 원문 문장·표·이미지 복제는 없습니다.
- 수정 내용: 목차를 `데이터의 뜻과 종류`, `데이터를 모으고 정리하는 방법`, `데이터를 읽을 때 확인할 점`으로 재구성했습니다. 본문은 기록된 자료, 숫자/글/사진/소리 데이터, 관찰·조사, 표와 그래프, 출처와 조사 대상, 개인정보 관리로 흐름을 다듬었습니다. 직접 관련성이 약한 `marie-curie` 관련 지식 연결은 `generative-ai`로 교체했습니다.

### 3. algorithm

- id: `algorithm`
- 파일: `data/source/knowledge/digital/algorithm.json`
- 판단: 부분 수정
- 본문 판단: 뜻과 예시는 적절했지만 `문제를 푸는 순서`가 목차와 소제목에 반복되고, 일부 문장이 안내형으로 흘러 백과 문장으로 다듬을 필요가 있었습니다.
- 퀴즈 판단: 알고리즘의 핵심과 조건의 중요성은 유지하되, 추천 알고리즘 문항은 “태도”보다 실제 추천에 쓰이는 자료를 묻는 문항이 더 적절하다고 판단했습니다.
- 출처 판단: 국가교육과정정보센터, 한국지능정보사회진흥원, NIST 자료는 유지했습니다. 알콰리즈미 관련 설명은 NIST 출처 설명 범위 안에서 학생용 문장으로 정리했습니다.
- 수정 내용: 목차를 `알고리즘의 뜻`, `알고리즘이 쓰이는 곳`, `알고리즘 결과가 달라지는 까닭`으로 재구성했습니다. 검색·추천·계산·코딩·AI와의 연결을 사실 중심으로 정리하고, 퀴즈 선택지를 더 구체적인 오답으로 바꾸었습니다.

### 4. generative-ai

- id: `generative-ai`
- 파일: `data/source/knowledge/digital/generative-ai.json`
- 판단: 부분 수정
- 본문 판단: `AI` 항목과 확인 기준 설명이 중복되고, `책임 있게 쓰기`처럼 안내형 목차가 강했습니다. 생성형 AI 고유의 결과물 생성 방식과 사용 장면이 더 앞에 나와야 한다고 판단했습니다.
- 퀴즈 판단: 첫 문항은 유지하되 오답이 너무 동떨어진 표현이 되지 않도록 조정했습니다. 두 번째 문항은 “태도”보다 사실과 출처 확인을 묻는 방식으로 바꾸었습니다.
- 출처 판단: 한국지능정보사회진흥원과 개인정보보호위원회 자료는 유지했습니다. 기존 연표의 GAN, 트랜스포머, 대화형 생성 AI 확산 설명은 현재 출처 목록과 직접 대응이 약해 제외했습니다.
- 수정 내용: 목차를 `생성형 AI의 뜻`, `생성형 AI가 쓰이는 장면`, `생성형 AI 결과물의 확인 기준`으로 정리했습니다. 본문은 글·그림·소리·코드 생성, AI 이미지와 음성, 사실·출처·저작권·개인정보 확인 기준으로 재구성했습니다.

- 출처 판단: 한국언론진흥재단 미디어아카데미와 뉴스 리터러시 자료는 유지했습니다. 원문 문단이나 자료 복제는 없습니다.
- 수정 내용: 목차를 `가짜뉴스의 뜻`, `가짜뉴스가 퍼지는 방식`, `가짜뉴스 확인 기준`으로 재구성했습니다. 관련 지식에는 생성형 AI와 저작권을 추가해 AI 이미지·허위 정보·저작권 흐름을 더 자연스럽게 연결했습니다.
